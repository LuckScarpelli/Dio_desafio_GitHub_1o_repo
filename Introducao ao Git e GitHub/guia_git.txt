echo > "..." --> printa msg no terminal ou cria arquivo

Ciclo de vida (Git)
Untracked --> git add (mover arquivo direto para Staged)
Tracked
    Unmodified --> arquivo que ainda nao foi modificado comparando sha1, se alterar com git add vai direto para Staged
    Modified --> roda git add retorna para unmodified
    Staged --> pronto para ser carregado no commit e retorna o arquivo para modified como snapshot.

Servidor --> repositório remoto
